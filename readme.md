## 2. 介绍

- 该项目用于 vue3的快速使用

## 3. 环境

编辑器

- `VSCode`

依赖

- `node.js` >= 18.16.0
- `npm`
- `pnpm`

框架

- `vue3`

## 4. 使用
下载依赖

```sh
pnpm install
```

启动
```sh
pnpm dev
```

构建

```sh
pnpm build
```
